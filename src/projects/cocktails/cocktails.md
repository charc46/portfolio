Mr Cocktails World is a Rails app that allows you to add your favourite cocktail to the list! Including a title, ingredients and a photo.

## Technologies used:
1. Ruby on Rails
2. HTML
3. CSS/Sass
4. PostgreSQL
5. Hosted on Heroku
6. Image hosting with Cloudinary
